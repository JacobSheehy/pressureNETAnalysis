pressureNET Live API Documentation
================================================
The pressureNET Live API can be used to access the database of pressure data that pressureNET has collected. This documentation describes this HTTPS API in detail, with examples.

Use of the API requires an API key, which you need to `sign up for <http://pressurenet.cumulonimbus.ca/customers/livestream/>`_.
